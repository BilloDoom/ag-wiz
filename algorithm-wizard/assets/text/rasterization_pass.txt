[font_size=16][b]Overview[/b][/font_size]
The rasterization pass converts [b]vector geometry[/b] (triangles) into [b]discrete pixels[/b] (fragments). This is the core of real-time rendering.

[font_size=16][b]What Happens[/b][/font_size]
[color=#88aaff]1.[/color] [b]Primitive Assembly[/b]
   Vertices are grouped into triangles based on index buffers.

[color=#88aaff]2.[/color] [b]Clipping[/b]
   Triangles outside the view frustum are discarded or clipped.

[color=#88aaff]3.[/color] [b]Rasterization[/b]
   Each triangle is converted to a set of [b]fragments[/b] (potential pixels):
   [code]Triangle → Fragment Grid → Pixel Coverage[/code]

[color=#88aaff]4.[/color] [b]Fragment Shader Execution[/b]
   For each fragment, the GPU calculates:
   • [color=#ff8888]Base Color[/color] - Material albedo/diffuse
   • [color=#ff8888]Lighting[/color] - Direct and indirect illumination
   • [color=#ff8888]Texturing[/color] - Sampled texture colors
